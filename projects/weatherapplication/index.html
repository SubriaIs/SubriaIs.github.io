Weather Application

Overview


Objective: A cross-platform React Native application that provides real-time weather data, a 7-day forecast for Tampere, location-based weather retrieval, and a map view with latitude/longitude details, using the OpenWeatherMap API, with Celsius/Fahrenheit toggle options.


Duration: April 2025 - June 2025


Team Size: Solo project

Tech Stack



Frontend: React Native


Backend: OpenWeatherMap API


Design Tools: React Native Paper


Key Features


• Current weather information: Temperature, wind and weather description are displayed.


• Celsius/Fahrenheit toggle option: User can view the temperature in Celsius/Fahrenheit.


• Tampere weather forecast: Provide upcoming 7-days weather information of Tampere.


• Weather Icons: Base on current weather, weather icons are displayed.


• Latitude/Longitude: Display the latitude/longitude information of the user current location.


• Map view: User can see the current location in the map view.


Achievements


- Up-to-date weather information are provived to the user.

- Develope user friendly interface.

- Ensured the app support Cross-Platform.
