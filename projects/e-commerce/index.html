E-Commerce

Overview


Objective: The goal of this project was to develop a robust E-commerce platform to provide a seamless shopping experience for users. The application allows users to browse products, place orders, and manage their profiles. Admins can manage product catalogs, categories, and orders, while the system provides secure payment processing and order tracking features.


Duration: February 2025 - April 2025


Team Size: Solo project

Tech Stack


• Backend: Java (JAX-RS), JPA (Java Persistence API), Hibernate, MySQL, BCrypt, Docker


• Frontend: React & TypeScript, Redux, Axios, Material-UI, Docker


• Deployment: Docker Compose

Key Features


• User Authentication: Users can register, login, and manage their profiles. Passwords are securely hashed using BCrypt, and tokens are used for maintaining user sessions.


• Product Management (Admin): Admin users can add, update, and delete products, manage product details (e.g., name, description, price).


• Category Management (Admin): Admins can add, delete, product categories, ensuring products are grouped correctly for easier navigation by customers.


• Order Management: Users can place orders, select products, specify quantities, and view the total cost of their order. Admins can track orders through various statuses (e.g., Pending, Shipped, Delivered, Cancelled).


• Payment Processing and PDF Payment Receipts: A downloadable PDF receipt is generated after confirming the order.


• Order Tracking: Users can track the status of their orders (e.g., Pending, Processing, Shipped, Delivered), ensuring they stay updated on their purchase’s progress.


• Product Search & Filtering: Users can search for products by name, category and price ranges.


• Responsive User Interface: The frontend is built with React and Material-UI, ensuring a responsive and user-friendly design across both desktop and mobile devices.


• Custom Exception Handling: Custom exception handling through SLServiceException and GenericExceptionMapper to handle errors efficiently and provide meaningful error messages to the user.


• State Management: Redux is utilized to manage the global application state, ensuring smooth data flow.

Achievements


• Secure User Authentication: Implemented BCrypt for password security and tokens for session management, ensuring secure login processes for users.


• Efficient Database Management: Leveraged JPA and Hibernate for ORM and simplified database interactions with MySQL, providing a robust backend.


• Admin Features: Created an intuitive admin panel allowing administrators to manage products, categories, and orders efficiently, offering real-time updates and easy control.


• Responsive UI/UX: Developed with React and Material-UI, the app provides a seamless user experience on both web and mobile platforms.


• Dockerized Deployment: Utilized Docker for both frontend and backend deployment, simplifying environment management and enhancing scalability.
