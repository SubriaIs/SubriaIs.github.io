Weather Data Visualization

Overview


Objective: This application provides real-time weather information for Tampere, including temperature, feels-like temperature, wind speed, humidity, and general conditions. In addition to the current weather, it displays detailed environmental data collected from sensors, such as temperature, wind speed, light levels, rainfall, and humidity. Users can explore the latest measurements in table format, filter data by specific time ranges or dates, and gain insights through bar and line chart visualizations.


Duration: March 2024 - May 2024


Team Size: Solo project

Tech Stack


• technologies: HTML5, JavaScript, Css

Key Features


• Real-Time Weather Display: View up-to-date weather information for Tampere, including temperature, feels-like temperature, wind speed, humidity, and sky conditions.


• Environmental Data Table: Access the latest environmental measurements such as temperature, wind speed, rainfall, humidity, wind direction, and light levels in tabular format.


• Search and Filter Options: Filter data by predefined time ranges (recent, 24h, 48h, 72h, 1 week) or search by specific date to analyze historical data.


• Temperature and Wind Speed Views: Explore the latest 20 readings of temperature and wind speed with corresponding search functionality and visualizations.


• Visual Analytics: Interpret data trends easily using integrated bar and line charts for temperature, wind speed, and other metrics.


• Multi-Measurement UI: View multiple environmental parameters in one interface, with selectable measurement types and time intervals.

Achievements


• Integrated Real-Time Weather Data: Successfully implemented real-time weather display for Tampere city with accurate and timely updates.


• Sensor-Based Data Visualization: Incorporated real environmental sensor data into the application, enhancing relevance and reliability.


• Searchable Historical Data: Enabled users to filter and retrieve weather data across various time intervals and specific dates.


• Interactive Visualizations: Developed intuitive bar and line chart visualizations for easy interpretation of temperature, wind speed, and other environmental trends.


• User-Friendly Interface: Designed a clean, accessible UI with clear data presentation and responsive components for enhanced user experience.


• Comprehensive Multi-Metric Support: Provided views for multiple environmental parameters, such as rainfall, light, and wind direction, in both visual and tabular formats.
