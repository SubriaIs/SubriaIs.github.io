Customer Support Management

Overview


Objective: This is a console-based Hotel Room Booking System implemented in C++. It allows users to book rooms, manage reservations, and calculate billing. The system supports searching reservations by number or name and uses a stack data structure to manage booking records dynamically. The program also features automated reservation number generation and differentiated room types with availability checking.


Duration: November 2022 - December 2022


Team Size: Solo project

Tech Stack


Language: C++


Libraries: Standard C++ Libraries

Key Features


• Struct-Based Reservation System: Efficiently manages room bookings using structured data types (Room and Reservation) and a stack to store reservations.


• Room Selection & Availability: Users can choose from single or double rooms, with real-time availability checks.


• Easy Search: Quickly find bookings by reservation number or reserver name.


• Auto-Generated Reservation IDs: Unique booking numbers are created automatically.


• Instant Billing: Generates an invoice with discount logic after reservation.


• User-Friendly Menu: A simple menu interface allows users to book, search, and view reservations seamlessly.


Achievements


- Designed a fully functional hotel booking system using C++ and STL.


- Implemented flexible search options for fast retrieval of reservations.


- Created a stack-based system to simulate real-world reservation handling.


- Handled pricing logic with discounts and generated invoices.


- Applied structured programming principles for modular and maintainable code.
