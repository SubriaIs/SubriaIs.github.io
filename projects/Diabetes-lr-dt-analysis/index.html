Diabetes Prediction with Logistic Regression and Decision Tree

Overview


Objective: This project aims to predict the likelihood of diabetes using two supervised machine learning models—Logistic Regression and Decision Tree—on a real-world diabetes dataset collected from the National Institute of Diabetes and Digestive and Kidney Diseases. The dataset contains 768 instances with 9 numerical health-related features.

Key steps in the project include:

• Data Preprocessing: Replacing invalid zero values with mean values, splitting data into training and testing sets (80/20 split), and standardizing features using StandardScaler.

• Model Development:
• Logistic Regression: Trained and evaluated using accuracy, precision, recall, and confusion matrix metrics.
• Decision Tree: Trained with a maximum depth of 3, visualized using Graphviz, and evaluated using the same metrics for comparison.

• Model Evaluation: Logistic Regression demonstrated better performance in accuracy, precision, and recall compared to the Decision Tree model.

• Prediction on New Data: Both models were used to predict diabetes outcomes for new patient profiles, showcasing their practical application.

• Conclusion: Logistic Regression outperformed the Decision Tree in this dataset, though further improvements could be achieved through advanced preprocessing and hyperparameter tuning.

Key Features


• Real-world Medical Dataset:
-Collected from the National Institute of Diabetes and Digestive and Kidney Diseases
-Contains 768 patient records with 9 numerical health-related attributes


• Data Cleaning & Preprocessing:
- Replaced invalid zero values in critical columns (e.g., Glucose, BMI, Insulin) with column means
- Applied feature scaling using StandardScaler
- Data split into 80% training and 20% testing sets


• Two Supervised ML Models:
- Logistic Regression for linear classification
- Decision Tree Classifier with depth constraint and visual interpretation


• Performance Evaluation:
- Metrics used: Accuracy, Precision, Recall, Confusion Matrix
- Logistic Regression demonstrated higher accuracy and precision than Decision Tree


• Prediction on New Data:
- Both models used to predict outcomes on new, unseen patient data


• Tools & Libraries:
- Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn, Graphviz

Achievements


• Successfully implemented and compared two supervised learning models—Logistic Regression and Decision Tree—to predict diabetes using a real-world medical dataset.

• Achieved 82% accuracy with Logistic Regression, outperforming the Decision Tree model (76%).

• Effectively handled data quality issues by replacing invalid zero values with column means and standardizing features to enhance model performance.

• Visualized Decision Tree structure using Graphviz, aiding in interpretability of model decisions.

• Used confusion matrix, precision, recall, and accuracy metrics to evaluate and compare both models thoroughly.

• Predicted diabetes outcomes on new patient data, demonstrating real-world applicability and inference capability.

• Showcased practical use of machine learning in healthcare diagnostics through structured modeling and evaluation.
