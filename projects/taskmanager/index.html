Task Manager

Overview


Objective: This Feature-Rich Task Management Application, built with React, helps users efficiently manage their tasks using advanced features like tagging, time-based filtering, and interface customization. Users can create, edit, and remove tasks, assign tags for better organization, and activate tasks or tags to track time spent. The app includes two working modes (default and alternative), allowing either multiple or single task activations. A customizable UI with color themes enhances the user experience, making the app both functional and visually adaptable.


Duration: September 2023 - November 2023


Team Size: Solo project

Tech Stack


• Backend: JSON Server


• Frontend: React.js, react-select/creatable, react-drag-reorder, moment.js


• Deployment Tools: Node.js, npm

Key Features


• Home View:
- View and manage all tasks and tags.
- Filter tasks by selecting specific tags.


• Task Management:
• Add New Task: Create new tasks and assign existing or newly created tags.
• Edit Task: Modify task names and add or remove tags.
• Remove Task: Easily delete tasks with a single click.
• Activate Tasks/Tags: Activate tasks and individual tags using the "Active" button.
• Deactivate Tasks/Tags: deactivate tasks and individual tags using the "Deactive" button.


• Filter Tasks:
• Filter by Time: Filter tasks based on specific start and end times, showing active tasks within that interval.
• Total Active Time: View the total active time for individual tasks or tags within the selected time frame.


• Settings:
• Choose Color: Customize the app's user interface by selecting a preferred color.
• Choose Mode:
• Default Mode: Multiple tasks can be active simultaneously.
• Alternative Mode: Only one task can be active at a time.

Achievements


• Intuitive User Interface: Successfully designed and implemented an easy-to-use UI that enables efficient task and tag management.


• Customizable User Experience: Implemented color theme and mode selection, giving users full control over the app’s appearance.


• Seamless Task Filtering: Enabled advanced time-based filtering, allowing users to view tasks and track their active time over custom intervals.


• Efficient Task Management: Developed features for adding, editing, removing, and activating tasks and tags with just a few clicks.


• Flexible Task Activation: Created two modes (Default and Alternative) for task activation, offering users more control over their workflow.


• Responsive Design: Ensured the app is fully responsive, offering a smooth experience across all devices and screen sizes.


• Backend Integration: Successfully integrated JSON Server for a lightweight backend solution, enabling dynamic task storage and retrieval.
