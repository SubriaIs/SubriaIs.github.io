Shopping List App

Overview


Objective: A real-time collaborative shopping list application that allows users to create, manage, and share shopping lists with others. With built-in group management and instant notifications, users can seamlessly coordinate purchases and keep everyone on the same page.


Duration: October 2024 - December 2024

Tech Stack


Frontend: React, Redux, TypeScript


Backend: Java, JAX-RS (Java RESTful services), JPA, Hibernate


Database: Relational database (MySQL)


Key Features


User Management: Sign up, log in, update password, and view profile


Shopping List Management: Create, edit, delete shopping lists with product-level details


Group Collaboration: Auto-created groups per list, member management, and list sharing


Real-Time Notifications: Automatic alerts for list updates, member changes, and product edits


Product Control: Add/remove/edit products with quantity, unit, and purchased status


UI/UX Highlights: Edit/View modes, searchable lists/products, auto-sync and manual refresh

Achievements


• Enabled real-time collaborative shopping with live updates and notifications


• Simplified user experience with intuitive list and member management


• Designed with modular, scalable architecture using modern frontend and backend technologies


• Improved productivity for users by automating group creation and syncing updates seamlessly
