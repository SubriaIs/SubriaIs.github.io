Shopping List App

Overview


Objective: This feature-rich Shopping List Application enables users to efficiently create, edit, and manage personal and shared shopping lists. It supports user authentication, real-time notifications, profile management, and collaborative list sharing. The app automatically syncs updates, allows product-level management within lists, and integrates with a backend API for full-stack functionality. Whether accessed locally or via Docker, the application provides a smooth and intuitive user experience for individuals and groups managing shopping needs.


Duration: October 2024 - December 2024


Team Size: Solo project

Tech Stack


Frontend: React, Redux, TypeScript


Backend: Java, JAX-RS, JPA, Hibernate


Database: Relational database (MySQL)


Containerization/Deployment: Docker


Key Features


• User Account Management:
• Sign Up: Create a new account by providing username, email, phone number, and password.
• Login: Securely log in using registered credentials.
• Update Password: Change password from the profile section.


• Shopping List Management:
• Create List: Add a new shopping list with a name and description.
• Edit List: Modify list name, description, and product details.
• View All Lists: Access and search through all personal and shared shopping lists.
• View Specific List: See full details, including creator, products, and members.
• Delete List: Permanently remove a shopping list with confirmation.


• Product Management within Lists:
• Add/Remove Products: Manage products with fields like name, quantity, purchased status, and unit.
• Product Filtering: Filter products by name within a list.


• Sharing & Collaboration:
• Share Lists: Invite other registered users to collaborate on lists.
• Manage Shared Lists: View, edit, or delete lists shared by others.
• Group Membership: Automatically managed groups for each shared list.


• Notifications:
• Real-Time Updates: Get notified of list changes, member actions, and product updates.
• Notification Center: Access all alerts via the bell icon.


• User Profile:
• View and manage profile details including email, username, and phone number.


• Auto-Sync & Manual Refresh:
• Auto-Sync: Lists automatically update every 30 seconds.
• Refresh Now: Manually reload data instantly.


• Modes:
• Edit Mode: Modify lists and products.
• View Mode: Read-only mode for reviewing list contents.


• Logout:
• Securely log out of the application at any time.

Achievements


• Full User and Group Management: Implemented robust user registration, authentication, and automatic group creation/removal tied to shopping lists for seamless collaboration.


• Collaborative List Sharing: Enabled real-time multi-user collaboration on shopping lists with dynamic member management and access control.


• Real-Time Notification System: Integrated an automated notification system that informs users of all relevant activities like product changes and member updates without manual input.


• Product-Level Control: Developed detailed product handling features (add, edit, delete, filter) within each shopping list, enhancing list usability.


• Flexible Deployment Options: Delivered support for both local development and Docker-based deployment, allowing easy setup across environments.


• Auto-Sync and Manual Refresh: Implemented automatic list updates every 30 seconds and a manual refresh feature for immediate synchronization.


• Secure and Scalable Architecture: Designed a clean separation between frontend and backend with token-based authentication for secure API access.


• Intuitive UI/UX: Created a clean, user-friendly interface with clear navigation between personal lists, shared content, and profile settings.
