Diabetes Prediction with ANN and Logistic Regression

Overview


Objective: This project focuses on predicting diabetes using two supervised machine learning models—Artificial Neural Network (ANN) and Logistic Regression—on a real-world medical dataset collected from the National Institute of Diabetes and Digestive and Kidney Diseases. The dataset contains 768 instances with 9 numerical features related to patient health metrics.

Key steps in the project include:

• Data Preprocessing: Replacing invalid zero values with mean values, splitting data into training/testing sets, and scaling features using StandardScaler.

• Model Development:
• ANN Model: Built using Keras Sequential API with ReLU and Sigmoid activations, evaluated with confusion matrix, accuracy, precision, and recall. Dropout layers were added to reduce overfitting.
• Logistic Regression: Trained and evaluated using the same metrics for performance comparison.

• Model Evaluation: Logistic Regression achieved slightly better accuracy, precision, and recall compared to the ANN model.

• Prediction on New Data: Both models were saved and later used to predict outcomes on new patient data using consistent scaling and thresholding.

• Conclusion: Logistic Regression outperformed ANN in this case, but ANN performance could be improved with further tuning and regularization.

Achievements


• Successfully built and compared two supervised learning models (ANN & Logistic Regression) to predict diabetes using a real-world dataset.

• Achieved 82% accuracy with Logistic Regression, outperforming the ANN model (79.87%).

• Implemented effective data preprocessing, including handling invalid values and applying feature scaling to improve model performance.

• Addressed overfitting in ANN by introducing Dropout layers, improving generalization capability.

• Saved and reused models and scalers for future predictions on unseen patient data—enabling scalable deployment.

• Visualized key evaluation metrics (confusion matrix, loss curves, etc.) to understand and interpret model performance clearly.

• Demonstrated practical application of machine learning in healthcare by accurately predicting diabetes risk.
